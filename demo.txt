#Quick setup — if you’ve done this kind of thing before
Get started by creating a new file or uploading an existing file. We recommend every repository include a README, LICENSE, and .gitignore.

link: 

creating a new file: https://github.com/kevinhotmail/git-basics/new/main
uploading an existing file: https://github.com/kevinhotmail/git-basics/upload


#or create a new repository on the command line

echo "# git-basics" >> README.md
git init
git add README.md
git commit -m "first commit"
git branch -M main
git remote add origin https://github.com/kevinhotmail/git-basics.git
git push -u origin main
